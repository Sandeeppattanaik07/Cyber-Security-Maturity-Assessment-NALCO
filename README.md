# Cyber Security Maturity Assessment (NALCO)

This project presents a comprehensive cybersecurity maturity assessment conducted during my internship at NALCO, based on industry-standard frameworks.

---

## 📌 Objective
To evaluate the organization’s cybersecurity posture, identify gaps, and recommend improvements aligned with global security standards.

---

## 🛠 Frameworks Used
- NIST Cybersecurity Framework (CSF)
- ISO/IEC 27001 (Annex A Controls)

---

## 📊 Methodology
- Assessed 15 key security domains (Access Control, Network Security, Incident Response, etc.)
- Performed domain-wise maturity evaluation
- Conducted gap analysis and risk prioritization
- Mapped controls against ISO 27001 and NIST CSF

---

## 📈 Key Findings
- Overall maturity level: **3.58 (Defined)**
- Identified gaps in monitoring, logging, and incident response processes
- Highlighted areas requiring stronger access control and security governance

---

## 🚀 Recommendations
- Implementation of **SIEM (Security Information and Event Management)**
- Adoption of **SOAR and XDR solutions**
- Strengthening endpoint security (EDR)
- Improving incident detection and response workflows
